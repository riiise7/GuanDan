# 检查分支
git branch
！！！非当前分支颜色是白色
# 新分支
git checkout -b branchname
# 换分支
git checkout branchname
# 合并分支
git merge branchname
！！！如果你希望branch1=branch1合并branch2，那就先切换到branch1，然后输入git merge branch2。
# 删除分支（本地）
git branch -d branchname
# 删除分支（远程）
git push origin --delete branchname
